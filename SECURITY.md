  # Security Policy

  This is a personal portfolio site at https://yoel.online. Source code is open,
  but the deployment runs API keys server-side via Netlify env vars.

  ## Reporting a Vulnerability

  Kalau lo nemu hal yang sensitif (kebocoran key, XSS, request smuggling,
  prompt injection di Talkative Yoel, atau apapun yang bisa di-exploit),
  email langsung:

    yoelandreasmanoppo@gmail.com

  Subject: `security: <one line summary>`

  Gue baca dalam 24 jam, mostly lebih cepet. Jangan post public issue dulu
  sampai gue konfirmasi. Kalau ternyata bener-bener bug, gue credit lo di
  commit / changelog kalau lo mau.

  Bukan bug bounty program — appreciation only.
